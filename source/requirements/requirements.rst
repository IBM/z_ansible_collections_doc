=====================
Software requirements
=====================

Before you install any IBM z/OS® collection, you must configure a **control node** and a **managed node** with a minimum set of requirements. 

Each IBM z/OS collection, or each version of a certain collection have different requirements ordependencies on control nodes and managed nodes. To find more details, check the following links for the specific collection.
