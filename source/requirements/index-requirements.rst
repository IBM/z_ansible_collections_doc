=============
Prerequisites
=============

Before you install any IBM z/OS® collection, you must configure a **control node** and a **managed node** with a minimum set of requirements. 

Each IBM z/OS collection, or each version of a certain collection may have different requirements or dependencies for control nodes and managed nodes. To find more details, check the following support matrix for the specific collection.
